\section*{Построение арбитражного портфеля в однопериодической модели}

\subsection*{Исходные данные}

\begin{itemize}
    \item Цена безрисковой облигации: $B_0 = 10$
    \item Безрисковая процентная ставка: $r = 10\%$
    \item Цена активов:
    \begin{itemize}
        \item $S_1 = 55$
        \item $S_2 = 22$
    \end{itemize}
    \item Дивиденды:
    \begin{itemize}
        \item $d_1 = 20\%$
        \item $d_2 = 50\%$
    \end{itemize}
\end{itemize}

Будущая стоимость облигации:
\[
B_1 = B_0 (1 + r) = 10 \cdot 1.1 = 11
\]

\subsection*{Будущие цены активов}

\textbf{Благоприятный сценарий (Up):}
\[
S_1^u = 55 \cdot 1.2 = 66, \quad S_2^u = 22 \cdot 1.5 = 33
\]

\textbf{Неблагоприятный сценарий (Down):}
\[
S_1^d = 66 \cdot \frac{1}{2} = 33, \quad S_2^d = 33 \cdot \frac{1}{3} = 11
\]

\subsection*{Построение реплицирующего портфеля}

Ищем портфель $(x_1, x_2)$, реплицирующий облигацию:
\[
\begin{cases}
66x_1 + 33x_2 = 11 \\
33x_1 + 11x_2 = 11
\end{cases}
\]

Решение системы:

\[
(66x_1 + 33x_2) - (33x_1 + 11x_2) = 0
\]
\[
33x_1 + 22x_2 = 0 \Rightarrow 3x_1 + 2x_2 = 0 \Rightarrow x_1 = -\frac{2}{3}x_2
\]

Подставим:
\[
33\left(-\frac{2}{3}x_2\right) + 11x_2 = 11
\]
\[
-22x_2 + 11x_2 = 11 \Rightarrow -11x_2 = 11 \Rightarrow x_2 = -1
\]

\[
x_1 = \frac{2}{3}
\]

\subsection*{Стоимость портфеля}

\[
V_0 = \frac{2}{3} \cdot 55 - 1 \cdot 22 = 36.67 - 22 = 14.67
\]

\subsection*{Вывод об арбитраже}

Реплицирующий портфель даёт выплату:
\[
V_1 = 11
\]

Однако:
\[
V_0 = 14.67 > B_0 = 10
\]

Следовательно, облигация недооценена, и существует арбитраж.

\subsection*{Арбитражная стратегия}

\begin{itemize}
    \item Купить 1 облигацию
    \item Продать портфель:
    \begin{itemize}
        \item $\frac{2}{3}$ актива $S_1$
        \item $-1$ актива $S_2$ (то есть купить 1 единицу)
    \end{itemize}
\end{itemize}

\subsection*{Начальный доход}

\[
\Pi_0 = 14.67 - 10 = 4.67
\]

\subsection*{Выплаты в будущем}

\[
\Pi_1 = 11 - 11 = 0
\]

\subsection*{Заключение}

Существует арбитражная возможность:
\begin{itemize}
    \item Безрисковая прибыль: $4.67$
    \item Будущие выплаты: $0$
\end{itemize}
